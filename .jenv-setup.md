# jenv 配置说明

本项目已配置使用 jenv 管理 Java 版本。

## 配置信息

- **项目 Java 版本**: 17（通过 `.java-version` 文件设置）
- **已安装的 JDK**:
  - Java 17.0.13
  - Java 1.8.0_431

## 使用方法

### 查看当前 Java 版本

```bash
java -version
mvn -version
```

### 切换 Java 版本（全局）

```bash
jenv global 17      # 设置为 Java 17
jenv global 1.8     # 设置为 Java 8
```

### 切换 Java 版本（项目级别）

```bash
jenv local 17       # 为当前项目设置 Java 17
jenv local 1.8      # 为当前项目设置 Java 8
```

### 查看所有可用版本

```bash
jenv versions
```

### 添加新的 JDK

```bash
jenv add /path/to/jdk
```

## 注意事项

1. **首次使用**: 需要重新打开终端或运行 `source ~/.zshrc` 使配置生效
2. **项目级别**: 项目根目录的 `.java-version` 文件会自动设置项目使用的 Java 版本
3. **Maven 集成**: jenv 的 maven 插件已启用，Maven 会自动使用 jenv 设置的 Java 版本

## 验证配置

```bash
# 检查 Java 版本
java -version

# 检查 Maven 使用的 Java 版本
mvn -version

# 检查 jenv 状态
jenv versions
```
